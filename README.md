# KeepWiz

![Version](https://img.shields.io/badge/version-1.2.17-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Docker](https://img.shields.io/badge/docker-ready-brightgreen)

A full-stack application for tracking and managing Wizard and card game scores, player statistics.

## Features

- Track player stats and game history
- Player statistics
- MongoDB backend for authentication and real-time features
- User authentication and authorization
- Mobile-friendly responsive design with PWA support
